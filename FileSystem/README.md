1. Requirements

Our File System must support:

File Operations
Create file
Read file
Write/Overwrite file
Delete file
Track size & timestamps
Support text/binary types
Directory Operations
Create directory
Add/remove children
Recursive structure
List children
Permission Management
Read (r)
Write (w)
Execute (x)
Applied on both files & directories
Error Handling
Invalid permissions
File not found
Duplicate name
Directory not found
Scalability
Very large directory trees
Efficient search & traversal
Extendable for symbolic links, versioning, caching
